# Portfolio

A personal portfolio site built with Next.js and React. This repository contains the source for the portfolio site, demo projects, and site content.

## Quick start

Install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
pnpm start
```

## Project structure

- `app/` — Next 13 app directory pages and providers
- `components/` — UI components used across the site
- `public/` — static assets (images, fonts)
- `styles/` — global SCSS and variables
- `__tests__/` & `cypress/` — unit and end-to-end tests
